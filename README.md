# PatrickChat
Чат созданный усилиями группы

Специальная версия ко Дню святого Патрика

## Функциональность администратора

Первый подключившийся клиент автоматически получает права администратора.

### Доступные команды администратора:

- `/kick <id>` - Исключить клиента из чата по его ID
  - Пример: `/kick 2` - исключит клиента с ID 2
  - Все пользователи получат уведомление об исключении
  
- `/shutdown <секунды>` - Запланировать выключение сервера через указанное время
  - Пример: `/shutdown 10` - сервер выключится через 10 секунд
  - Пользователи будут получать уведомления с обратным отсчётом
  - Для времени <= 10 секунд уведомления приходят каждую секунду

### Запуск сервера

```bash
mvn clean compile
java -cp target/classes patrick.StartServer
```

### Подключение клиента

```bash
java -cp target/classes patrick.ClientCommucation <server_ip> <nickname>
```
